Как подключить принтер Epson L120 к ноутбуку: 3 рабочих способа + решение проблем

Подключение принтера Epson L120 к ноутбуку кажется простой задачей — пока не сталкиваешься с отсутствием драйверов, ошибками распознавания устройства или проблемами совместимости. Этот струйный МФУ, выпущенный в 2013 году, до сих пор остаётся популярным благодаря системе непрерывной подачи чернил (СНПЧ) и низкой стоимости печати. Однако его "возраст" создаёт нюансы: современные ОС (особенно Windows 11 и macOS Ventura/Sonoma) не всегда корректно определяют устройство автоматически.

В этой статье — 3 проверенных способа подключения (USB, Wi-Fi и сетевой кабель), актуальные драйверы для всех ОС, а также решения типичных ошибок вроде «Устройство не опознано» или «Драйвер не найден». Особое внимание уделим настройке Epson L120 на ноутбуках с ARM-процессорами (например, MacBook M1/M2 или Windows на Snapdragon), где стандартные методы часто не работают. Если вы уже пытались подключить принтер и получили ошибку — пропустите базовые шаги и переходите сразу к разделу «Устранение неполадок».

Требования перед подключением: что проверить заранее

Прежде чем соединять Epson L120 с ноутбуком, убедитесь, что выполнены минимальные условия:

  • 🔌 Кабель USB 2.0 Type-B (в комплекте принтера). Использование USB 3.0 или удлинителей может вызвать ошибки передачи данных.
  • 💻 Свободный порт USB на ноутбуке. На моделях с единственным портом (например, MacBook Air) потребуется хаб.
  • 📥 Драйверы для вашей ОС. Для Epson L120 актуальные версии:
    • Windows 10/11: Epson L120 Series Printer Driver v6.95 (от 2023 года)
    • macOS: Epson Printer Software 10.85 (поддерживает Sonoma)
    • Linux: пакет epson-inkjet-printer-escpr (через apt или dnf)
  • 📄 Бумага и чернила. Принтер может не инициализироваться, если в лотке нет бумаги или закончились чернила (даже при подключении).

Если вы подключаете принтер впервые после покупки, обязательно удалите транспортные фиксаторы из картриджей и выполните начальную заправку чернил через программу Epson Software Updater. Пропуск этого шага приводит к повреждению печатающей головки.

⚠️ Внимание: На ноутбуках с Windows 11 24H2 и новее стандартные драйверы Epson L120 могут блокироваться системой безопасности. В этом случае потребуется отключить проверку подписи драйверов (инструкция в разделе «Особенности Windows 11»).

Способ 1: Подключение через USB (самый надёжный)

Проводное соединение остаётся самым стабильным методом, особенно для старых принтеров вроде L120. Следуйте пошаговой инструкции:

  1. Выключите принтер и ноутбук. Подсоедините кабель USB Type-B к задней панели Epson L120 (порт рядом с разъёмом питания). Другой конец вставьте в USB-порт ноутбука.
  2. Включите принтер, затем ноутбук. Дождитесь, пока индикатор питания на принтере перестанет мигать (обычно 10–15 секунд).
  3. Откройте «Устройства и принтеры»:
    • В Windows: нажмите Win + R, введите control printers.
    • В macOS: перейдите в Системные настройки → Принтеры и сканеры.
  • Дождитесь автоматического распознавания. Если принтер не появился в списке — переходите к ручной установке драйверов (раздел «Установка драйверов»).

    • На Linux (Ubuntu/Debian) после подключения выполните в терминале:

    sudo apt update

    sudo apt install printer-driver-escpr

    Установка драйверов для Epson L120

    Без правильных драйверов принтер будет определяться как «неизвестное устройство» или вообще не отобразится в системе. Скачайте ПО с официального сайта Epson (раздел «Поддержка» → введите модель L120).

    Операционная система Название драйвера Размер файла Ссылка на скачивание
    Windows 10/11 (x64) Epson L120 Series Printer Driver 45 МБ Скачать
    macOS 12–14 Epson Printer Software 120 МБ Скачать
    Linux (Debian/Ubuntu) epson-inkjet-printer-escpr 15 МБ sudo apt install printer-driver-escpr

    Для Windows:

    1. Запустите скачанный файл (например, epson12345.exe).
    2. Выберите Типичная установка (не «Минимальная»!).
    3. Подключите принтер по USB только когда программа попросит (обычно на шаге 3 из 5).
    4. Перезагрузите ноутбук после завершения установки.

    Для macOS:

    1. Откройте скачанный .dmg-файл.
    2. Запустите Epson Software Installer.
    3. В разделе Дополнительные принтеры отметьте Epson L120.
    4. Подтвердите установку в Системных настройках → Конфиденциальность (разрешите доступ к принтерам).
    ⚠️ Внимание: На MacBook с чипом M1/M2 драйверы Epson L120 могут не устанавливаться через стандартный инсталлятор. В этом случае используйте альтернативный метод:
    1. Скачайте драйвер для macOS 10.15 (даже если у вас новее ОС).
    2. Откройте терминал и выполните: 
      sudo xattr -r -d com.apple.quarantine /Путь/к/установочному/файлу.app
    3. Запустите установку вручную.
    💡

    Если после установки драйверов принтер печатает "пустые" листы, проверьте уровень чернил в картриджах через утилиту Epson Status Monitor (входит в комплект ПО).

    Способ 2: Подключение по Wi-Fi (если есть адаптер)

    Epson L120 не имеет встроенного Wi-Fi, но поддерживает подключение через внешний адаптер Epson Wireless Adapter ELPAP10 (покупается отдельно). Если адаптер установлен, следуйте инструкции:

    1. Подключите адаптер к USB-порту принтера (не ноутбука!).
    2. Нажмите кнопку Wi-Fi на адаптере на 3 секунды, пока индикатор не начнёт мигать.
    3. На ноутбуке:
      • В Windows: откройте Панель управления → Устройства и принтеры → Добавить принтер → Добавить сетевой принтер.
      • В macOS: Системные настройки → Принтеры → Добавить (+) → IP.
  • Введите IP-адрес принтера (узнать его можно, распечатав тестовый лист с настройками сети через панель управления Epson L120).

    • Если принтер не находится в сети:

    • 🔄 Перезагрузите роутер и принтер.
    • 📡 Проверьте, что ноутбук и принтер подключены к одной Wi-Fi сети (2.4 ГГц, не 5 ГГц!).
    • 🔒 Отключите брандмауэр/антивирус на ноутбуке (они могут блокировать соединение).
    Как распечатать тестовый лист с IP-адресом?

    Нажмите и удерживайте кнопку Отмена на принтере 5 секунд. Принтер распечатает лист с сетевыми настройками, где будет указан IP-адрес (например, 192.168.1.105).

    Способ 3: Подключение через сетевой кабель (LAN)

    Если у вашего Epson L120 есть порт Ethernet (на некоторых модификациях), можно подключить его напрямую к роутеру или ноутбуку через адаптер USB-to-Ethernet:

    1. Соедините принтер с роутером патч-кордом (LAN-кабель).
    2. На ноутбуке откройте Панель управления → Сетевые подключения и проверьте, что IP-адрес ноутбука и принтера находятся в одной подсети (например, 192.168.1.x).
    3. Добавьте принтер как сетевое устройство:
      • В Windows: Устройства и принтеры → Добавить принтер → Принтер не найден в списке → Добавить по TCP/IP-адресу.
      • В macOS: Системные настройки → Принтеры → Добавить → IP → Протокол: LPD.

    Для проверки соединения распечатайте тестовый лист через Панель управления принтера (нажмите и удерживайте кнопку Чёрный/Цветной 3 секунды).

    Устранение неполадок: что делать, если принтер не подключается

    Рассмотрим самые распространённые ошибки и их решения:

    Ошибка Причина Решение
    Устройство USB не опознано Проблемы с портом, кабелем или питанием
    • Попробуйте другой USB-порт (желательно 2.0).
    • Отключите другие USB-устройства (может не хватать питания).
    • Замените кабель (даже если внешне он целый).
    Драйвер не найден Устаревшее или отсутствующее ПО
    • Скачайте драйвер с официального сайта (не используйте диски из комплекта!).
    • В Windows обновите драйвер вручную через Диспетчер устройств.
    Принтер в очереди печати, но не печатает Зависание задания или ошибка spooler
    • Перезапустите службу печати: 
      net stop spooler

      net start spooler

    • Очистите очередь печати в Устройства и принтеры.

    Если принтер подключён, но печатает с полосами или пропусками:

    • 🖨️ Запустите чистку печатающей головки через Epson Printer Utility (3–5 циклов).
    • 🔧 Проверьте уровень чернил в картриджах (даже если они новые, могли быть неправильно заправлены).
    • 📄 Используйте бумагу плотностью 80 г/м² (тонкая бумага может застревать).
    ⚠️ Внимание: Если после подключения Epson L120 постоянно мигает оранжевой лампочкой, это указывает на переполнение памперса (впитывающей прокладки). В этом случае требуется сброс счётчика памперса через утилиту Adjustment Program. Самостоятельный сброс без очистки памперса приведёт к протечкам чернил!
    💡

    90% проблем с подключением Epson L120 решаются переустановкой драйверов или заменой USB-кабеля.

    Особенности подключения на Windows 11 24H2 и новее

    В последних версиях Windows 11 Microsoft ужесточила политику безопасности для устаревших драйверов. Если при установке вы видите ошибку «Этот драйвер заблокирован» или «Невозможно проверить издателя», выполните следующие шаги:

    1. Отключите обязательную проверку подписи драйверов:
      • Откройте Командная строка (администратор).
      • Выполните: 
        bcdedit.exe /set nointegritychecks on

        bcdedit.exe /set testsigning on

      • Перезагрузите ноутбук.
  • Установите драйвер вручную:
    • Откройте Диспетчер устройств.
    • Найдите Epson L120 с восклицательным знаком.
    • Выберите Обновить драйвер → Выполнить поиск на этом компьютере → Выбрать из списка → Установить с диска.
    • Укажите путь к распакованной папке с драйвером (файл .inf).
    • Верните настройки безопасности после установки: 
      bcdedit.exe /set nointegritychecks off

      bcdedit.exe /set testsigning off

    Если после этих действий принтер всё равно не работает, попробуйте подключить его через виртуальную машину с Windows 10 (например, VirtualBox или VMware) и настроить общий доступ к принтеру.

    Подключение к MacBook на чипе M1/M2 (ARM)

    Принтеры Epson L120 официально не поддерживаются на Apple Silicon (M1/M2), но есть обходные пути:

    1. Используйте Rosetta 2:
      • Установите драйвер для macOS 10.15 (Catalina).
      • Откройте терминал и выполните: 
        softwareupdate --install-rosetta
      • Запустите установочный файл драйвера через правый клик → Открыть с помощью Rosetta.
    2. Настройте общий принтер через другой компьютер:
      • Подключите L120 к ПК с Windows или Intel Mac.
      • Включите общий доступ к принтеру в настройках.
      • На MacBook M1/M2 добавьте принтер через Системные настройки → Принтеры → Добавить → Windows (укажите IP компьютера, к которому подключён принтер).
  • Используйте AirPrint-эмулятор:
    • Установите на ноутбук CUPS (система печати Unix) и настройте Epson L120 как AirPrint-устройство.
    • Инструкция: 
      sudo launchctl stop org.cups.cupsd

      sudo cupsctl --share-printers
      

      sudo launchctl start org.cups.cupsd

      • ⚠️ Внимание: На macOS Sonoma 14.4+ может потребоваться отключить защиту системы (csrutil disable в режиме восстановления), но это небезопасно для повседневного использования. Рекомендуем использовать общий принтер через другой компьютер.

      Частые вопросы по подключению Epson L120

      Можно ли подключить Epson L120 к ноутбуку через Bluetooth?

      Нет, Epson L120 не поддерживает Bluetooth. Единственные беспроводные варианты — Wi-Fi (с адаптером ELPAP10) или Ethernet (если есть порт).

      Принтер подключён, но печатает пустые листы. Что делать?

      Причины могут быть следующими:

      1. Закончились чернила — проверьте уровень через Epson Status Monitor.
      2. Засохли сопла печатающей головки — запустите чистку (3–5 циклов) через панель управления принтера.
      3. Неправильные настройки печати — в диалоге печати выберите Черновик вместо Высокое качество.
      4. Бракованные картриджи — попробуйте заменить картриджи (даже если они новые, могли быть повреждены при заправке).

      Если проблема остаётся — выполните глубокую чистку головки.

      Как подключить Epson L120 к ноутбуку на Linux (Ubuntu/Fedora)?

      На большинстве дистрибутивов Linux Epson L120 поддерживается через пакет epson-inkjet-printer-escpr:

      1. Установите драйвер: 
        sudo apt update

        sudo apt install printer-driver-escpr

        Для Fedora:

        sudo dnf install epson-inkjet-printer-escpr
      2. Подключите принтер по USB.
      3. Добавьте принтер через Настройки → Устройства → Принтеры.
      4. Если принтер не определяется, установите cups и перезапустите службу: 
        sudo systemctl restart cups

    Для сканирования установите sane-airscan или xsane.

    Можно ли использовать Epson L120 без установки драйверов?

    Технически да, но с серьёзными ограничениями:

    • В Windows принтер может определиться как Generic Text Only, но печать будет только в низком разрешении и без поддержки цвета.
    • В macOS/Linux без драйверов печать невозможна — система просто не увидит устройство.

    Для полноценной работы (печать фотографий, двухсторонняя печать, сканирование) установка официальных драйверов обязательна.

    Как подключить Epson L120 к ноутбуку через USB-хаб?

    Использование USB-хаба возможно, но с оговорками:

    • 🔹 Хаб должен быть с внешним питанием (пассивные хабы не обеспечат достаточный ток).
    • 🔹 Подключайте принтер к USB 2.0-порту хаба (не 3.0!).
    • 🔹 Избегайте хабов с большим количеством подключённых устройств (может не хватить мощности).

      • Если принтер не определяется через хаб, попробуйте подключить его напрямую к ноутбуку для первоначальной настройки.
